# Payment Integration System

## Overview
Enterprise-grade Payment Integration System 
built with Spring Boot Microservices + Stripe API

## Tech Stack
- Java 17
- Spring Boot 3.x
- MySQL 8.0
- Docker & Docker Compose
- AWS (EC2, RDS, ECR)
- Stripe Payment API
- GitHub Actions CI/CD

## Microservices
| Service | Port | Description |
|---------|------|-------------|
| payment-validation-service | 8081 | Validates payment requests |
| payment-processing-service | 8082 | Core payment processing |
| stripe-provider-service | 8083 | Stripe API integration |

## Architecture
- Microservices Architecture
- REST API communication
- Docker containerization
- CI/CD with GitHub Actions
- Cloud deployment on AWS
